# ספר המתכונים שלי - אתר מתכונים עם הקראה קולית

## תיאור הפרויקט
אתר ספר מתכונים אינטראקטיבי עם פונקציית הקראה קולית שמסייעת בזמן הבישול.

## מסכים
- **מסך התחברות** - כניסה עם שם משתמש וסיסמה
- **מסך הגדרות אישיות** - התאמת זמן המתנה בין שורות ומצב תצוגה
- **מסך רשימת מתכונים** - רשימה עם חיפוש והוספת מתכונים
- **מסך צפייה במתכון** - הצגת המתכון עם הקראה קולית

## משתמשים ראשוניים
- **admin** / סיסמה: 123456
- **משתמש** / סיסמה: 1234

## הוראות הפעלה

### 1. הפעלה מקומית
1. פתח את הקובץ `Pages/login.html` בדפדפן
2. התחבר עם אחד מהמשתמשים הראשוניים
3. התחל לגלוש באתר!

### 2. העלאה לאינטרנט (אפשרויות חינמיות)

#### GitHub Pages
1. צור repository חדש ב-GitHub
2. העלה את כל הקבצים
3. עבור ל-Settings > Pages
4. בחר Source: Deploy from a branch
5. בחר branch: main
6. האתר יהיה זמין ב: `https://[username].github.io/[repository-name]/Pages/login.html`

#### Netlify
1. עבור ל-netlify.com
2. גרור את התיקייה לאזור "Deploy"
3. האתר יהיה זמין מיד

#### Vercel
1. עבור ל-vercel.com
2. חבר את ה-GitHub repository
3. האתר יהיה זמין מיד
